# 💰 FinTrack — Finance Dashboard

A responsive finance dashboard to track income, expenses, and spending patterns — built with React, TypeScript, and Tailwind CSS.

🔗 [Live Demo](#) · [GitHub](https://github.com/BhavyaCharitha2005/finance-dashboard)

---

## Tech Stack

- **React + TypeScript** — UI with full type safety
- **Vite** — Fast build tool
- **Tailwind CSS** — Responsive styling
- **Zustand** — State management
- **Recharts** — Interactive charts
- **React Router DOM** — Navigation

---

## Features

- **Dashboard** — Summary cards, balance trend chart, spending breakdown
- **Transactions** — Search, filter, sort with full CRUD for Admin
- **Insights** — Savings rate, top spending category, monthly comparison
- **Role Based UI** — Viewer (read only) and Admin (full access)
- **Responsive** — Works across all screen sizes

---

## Getting Started
```bash
git clone https://github.com/BhavyaCharitha2005/finance-dashboard.git
cd finance-dashboard
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

---

## Role Switching

Use the dropdown in the top right header to switch between **Viewer** and **Admin** roles. Admin can add, edit and delete transactions.
